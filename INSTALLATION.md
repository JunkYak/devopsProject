# Jenkins Installation Guide

## Prerequisites
- Windows Operating System
- Java JDK (11 or later)
- Git for Windows
- Maven

## Jenkins Installation Steps
1. Download Jenkins LTS from the official Jenkins website.
2. Run the installer and follow the setup wizard.
3. Unlock Jenkins using the initial admin password.
4. Install recommended plugins.
5. Create an admin user.
6. Configure Jenkins URL.

## Tool Configuration
- Configure JDK in Jenkins Global Tool Configuration.
- Configure Git installation.
- Configure Maven installation.

Jenkins is now ready for CI job creation.
